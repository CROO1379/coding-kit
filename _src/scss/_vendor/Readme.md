
SCSS / Vendor
==================================================

このディレクトリのCSSは、外部で作られたCSSフレームワークや、JSプラグイン付属のCSSを管理します。
例えば、スライダーやモーダルウィンドウなどのCSSです。



用法
--------------------------------------------------

必要に応じて、globalもしくはページ個別のSCSSファイルからインポートしてください。
デザインのためのスタイルは、各ページのCSSから上書きします。

原則として、このディレクトリに配置したスタイルの直接編集は禁止します。
